# Tauri 桌面验收清单

本清单用于每次修改 Tauri 外壳、小窗布局、弹窗、滚动、通知或桌面相关行为后做手动验收。当前阶段不验收菜单栏、Dock 隐藏、置顶窗口或真实打包发布。

## 验收前准备

- [ ] 当前工作区没有未保存的重要改动。
- [ ] 执行 `npm run test` 通过。
- [ ] 执行 `npm run build` 通过。
- [ ] 执行 `cargo check` 通过。
- [ ] 如果 `5173` 被其他应用占用，先释放端口；如果是本项目 Vite 服务，可直接复用。

## 启动桌面窗口

```bash
npm run tauri:dev
```

- [ ] 命令能启动 Tauri 桌面窗口。
- [ ] 窗口标题显示“急急如律令”。
- [ ] 初始窗口接近 `390px × 620px`。
- [ ] 页面内容加载的是本项目，而不是其他占用 `5173` 的页面。
- [ ] 首次编译后，再次启动速度明显变快。

## 小窗基础布局

- [ ] 小窗下没有横向滚动条。
- [ ] 应用内标题区不挤占过多第一屏高度。
- [ ] “历史”和“设置”按钮可见且可点击。
- [ ] 当前计时模式显示清楚，但不喧宾夺主。
- [ ] 页面主体可以纵向滚动，滚动时没有内容被系统标题栏遮住。

## 填写页

- [ ] 默认显示 1 套执行意图。
- [ ] 可以添加到 3 套执行意图。
- [ ] 长情境性执行意图输入时，输入框和页面滚动正常。
- [ ] 校验错误显示后不会撑出横向滚动。
- [ ] 提交后能进入仪式台。

## 小窗仪式台

- [ ] 创建 1 套时显示 1 个香炉。
- [ ] 创建 2 套时显示 2 个香炉。
- [ ] 创建 3 套时显示 3 个香炉。
- [ ] 小窗下显示香炉舞台，不显示完整主祭台三栏布局。
- [ ] 香炉内线香根数与所选香数一致。
- [ ] 线香不会顶出香炉槽位或背景框。
- [ ] 点击未开始香炉可以打开开始确认弹窗。
- [ ] 烧香中、休息中、已完成状态文案能完整显示。
- [ ] 长执行意图摘要不会遮挡按钮或撑破布局。

## 弹窗

- [ ] 开始确认弹窗不会超出窗口。
- [ ] 开始确认弹窗内容过长时可以滚动到底部按钮。
- [ ] 单炷结束弹窗按钮显示完整。
- [ ] 休息结束弹窗按钮显示完整。
- [ ] 放弃本轮确认弹窗显示完整。
- [ ] 历史删除或清空确认弹窗显示完整。
- [ ] 弹窗打开时，背景内容不会横向溢出。

## 计时流程

- [ ] 点击开始后，倒计时正常刷新。
- [ ] 单炷结束后弹出休息或续香选择。
- [ ] 选择休息后进入休息倒计时。
- [ ] 休息结束后可以继续下一炷香。
- [ ] 多炷香会按第 1、2、3 炷依次推进。
- [ ] 所有香完成后进入复盘页。

## 系统通知

- [ ] 首次开始计时时，macOS 会请求通知权限，允许后继续当前流程。
- [ ] 一炷香计时结束时，系统通知显示“这一炷香已经烧完。”。
- [ ] 休息计时结束时，系统通知显示“休息结束，是否继续下一炷香？”。
- [ ] 开始下一段计时后，不会继续保留上一段的待触发通知。
- [ ] 放弃本轮、保存复盘或丢弃恢复轮次后，不会继续触发旧通知。
- [ ] 进行中轮次点击“保留并收起”后，如果当前段计时结束，系统仍能发出通知。

## 复盘与历史

- [ ] 复盘页在小窗下可以滚动到保存按钮。
- [ ] 复盘长文本不会撑出横向滚动。
- [ ] 保存后历史记录出现本轮记录。
- [ ] 历史面板可以滚动查看多条记录。
- [ ] 历史筛选、搜索、导入、导出按钮在小窗下不拥挤到不可用。
- [ ] 设置面板在小窗下可以完整滚动查看。

## 关闭与恢复

关闭与恢复的产品语义见 `docs/TAURI_CLOSE_RESTORE_NOTES.md`。

- [ ] 填写页已有草稿时点击窗口关闭按钮，会出现应用内确认弹窗。
- [ ] 草稿关闭确认中点击“继续填写”，窗口保持打开。
- [ ] 草稿关闭确认中点击“关闭窗口”，窗口关闭，重新启动后不恢复草稿。
- [ ] 仪式台进行中点击窗口关闭按钮，会出现应用内确认弹窗。
- [ ] 进行中关闭确认里点击“继续当前轮次”，窗口保持打开。
- [ ] 进行中关闭确认里点击“保留并收起”，窗口收起且计时继续。
- [ ] 重新启动后，如果有未完成轮次，会出现恢复提示。
- [ ] 点击恢复后能回到当前轮次。
- [ ] 点击丢弃后回到填写页，不写入历史。
- [ ] 保存复盘后重新启动，不会提示恢复旧轮次。

## 当前不验收

- [ ] 不验收 macOS 菜单栏入口。
- [ ] 不验收 Dock 隐藏。
- [ ] 不验收窗口置顶。
- [ ] 不验收全局快捷键。
- [ ] 不验收正式应用图标。
- [ ] 不验收真实 macOS 打包发布。
