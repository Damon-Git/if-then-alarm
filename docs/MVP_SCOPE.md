# MVP 范围

《急急如律令》的第一轮 MVP 目标是跑通从执行意图填写到本地历史保存的完整闭环。当前版本先作为 Web App 实现，代码结构保持简单，方便后续迁移到 Tauri 并做成 macOS 桌面小窗工具。

## 当前要做

- 用户可以创建 1-3 套执行意图。
- 每套执行意图包含 1 条情境性执行意图、0-3 条预防性执行意图、1-3 炷香。
- 情境性执行意图使用“当……就……”句式，并做轻校验。
- 预防性执行意图使用“如果……那么……”句式；未填写时跳过校验。
- 提交后进入仪式台，显示情境性符箓占位、香炉占位、预防性符箓占位。
- 点击情境性符箓后开始当前炷香倒计时。
- 单炷香结束后弹出休息或续香选择。
- 休息结束后提醒继续下一炷香。
- 全部香完成后进入复盘流程。
- 复盘保存到 localStorage。
- 刷新页面后仍能查看历史记录。
- 历史记录支持删除单条和清空全部。
- 仪式台和复盘页提供“放弃本轮”保护入口。
- 仪式台和复盘页刷新/关闭页面前给出浏览器提示。

## 当前不做

- 不做香炉开盖动画。
- 不做符箓燃烧动画。
- 不做线香燃烧动画。
- 不做烟雾动画。
- 不做复杂音效。
- 不接后端。
- 不做真实桌面打包。
- 不做当前轮次的完整 session 恢复。
- 不重新绘制复杂符箓、香炉、线香。
- 不宣称宗教、玄学或超自然功效。

## 第一优先级

当前阶段以完整业务流程和状态可靠性为第一优先级。视觉只使用清晰的占位元素，保证未来可以替换为真实符箓、香炉、线香素材。
