# 验收清单

每次功能迭代后，按本清单选择合适范围手动验收。当前开发计时使用开发常量：单炷香 10 秒，休息 5 秒。

这份文档是“完整验收清单”，不要求每次小改都全部跑完。日常自用版发布前优先执行 `docs/SELF_USE_REGRESSION_RUNBOOK.md`；涉及桌面、小窗、持久化、通知、视觉状态机或真实素材时，再按本文件扩大覆盖面。

## 验收分级

- **P0 必验**：会影响当前轮次、历史数据、桌面启动、小窗恢复、通知时点、复盘保存的改动。
- **P1 应验**：会影响主祭台视觉状态、符箓 hover、香炉盖子、线香状态、声音提醒、设置持久化的改动。
- **P2 抽验**：只影响文案、文档、轻微样式、非核心分析展示的改动。
- **暂不验收**：签名、公证、DMG、高密度真实粒子系统等发布和高成本视觉项。

## 烧香与休息

- [ ] 单炷香倒计时结束后弹出“这一炷香已经烧完。”。
- [ ] 还有下一炷香时，可以选择“立刻续香”。
- [ ] 还有下一炷香时，可以选择“休息 5 分钟”。

- [ ] 休息结束后弹出“休息结束，是否继续下一炷香？”。
- [ ] 点击“开始下一炷香”后进入下一炷香倒计时。
- [ ] 最后一炷香结束后，该套显示“已完成”。
- [ ] 所有任务完成后停留在完成祭台场景，不自动进入复盘页。
- [ ] 完整窗口所有任务完成后显示独立收尾入口，包含“本轮香尽”和“进入复盘”。
- [ ] 完整窗口完成态中，“放弃本轮”弱化为次要操作，不与“进入复盘”并排争夺主操作。
- [ ] 小窗完成态不显示“本轮香尽”或“进入复盘”，只显示闭盖香炉。
- [ ] 点击“进入复盘”后进入复盘页。
- [ ] 计时进行中刷新并恢复时，剩余时间按真实经过时间重新计算。

- [ ] 休息计时已在离开期间结束时，恢复后直接显示休息结束提醒。
- [ ] 放弃本轮确认弹窗打开时，当前倒计时冻结；取消放弃后继续计时。
