# 声音提醒规格

本文记录钟声提醒的产品边界。当前版本使用 Web Audio 生成临时钟声，不引入音频文件；后续再替换正式钟声素材。

## 当前状态

- 设置面板提供“声音提醒”开关。
- 默认关闭。
- 开关会保存到应用设置 `isSoundReminderEnabled`。
- 开启后，专注结束、休息结束和整轮完成会播放一次临时钟声。
- 关闭后不播放声音。

## 播放规则

- 每段专注结束最多响一次。
- 每段休息结束最多响一次。
- 整轮完成最多响一次。
- 不在专注或休息开始时响。
- 不与同一事件的其他声音重复播放。
- 用户关闭“声音提醒”后，所有钟声都不播放。
- 播放失败时静默跳过，不影响计时、弹窗和系统通知。

## 声音原则

- 声音以克制钟声为主，不做复杂音效。
- 音量应偏低，避免打断用户。
- 未来正式素材需要确认版权和可分发权限。
- macOS 勿扰、系统静音和通知设置的行为需要单独验收。

## 后续实现

后续可以用正式钟声素材替换当前 Web Audio 临时钟声。声音播放逻辑应继续独立于系统通知；系统通知负责提醒可见性，声音提醒只作为可关闭的辅助反馈。
