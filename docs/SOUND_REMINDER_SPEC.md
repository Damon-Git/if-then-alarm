# 声音提醒规格

本文记录钟声提醒的产品边界。当前版本只预留设置项，不播放声音；下一版再接入钟声素材或 Web Audio。

## 当前状态

- 设置面板提供“声音提醒”开关。
- 默认关闭。
- 开关会保存到应用设置 `isSoundReminderEnabled`。
- 当前不开启任何实际声音播放。

## 未来播放规则

- 每段专注结束最多响一次。
- 每段休息结束最多响一次。
- 整轮完成最多响一次。
- 不在专注或休息开始时响。
- 不与同一事件的其他声音重复播放。
- 用户关闭“声音提醒”后，所有钟声都不播放。

## 声音原则

- 声音以克制钟声为主，不做复杂音效。
- 音量应偏低，避免打断用户。
- 未来正式素材需要确认版权和可分发权限。
- macOS 勿扰、系统静音和通知设置的行为需要单独验收。

## 后续实现

下一版可以先接本地占位钟声或 Web Audio 简单钟声，再替换正式素材。声音播放逻辑应独立于系统通知；系统通知负责提醒可见性，声音提醒只作为可关闭的辅助反馈。
