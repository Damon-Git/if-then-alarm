# 主祭台背景素材

`background.png` 是当前主祭台背景 v1，用于提供空祭台环境、桌面承托关系和“三个香炉并排放在同一张桌子上”的空间基准。

当前使用原则：

- 背景只承担场景和桌面位置参考，不承载功能信息。
- 背景不能自带固定香炉、符箓、线香、烛台或文字。
- 香炉、线香和符箓仍由 React 组件分层渲染，不能烘焙进背景图。
- 预防性符箓需要贴近桌子前沿，并在 2-3 张时水平排列。
- 1-3 个香炉必须使用同一条水平基线，不能因为槽位数量或状态产生上下错位。
- 后续若继续迭代背景，优先保持相近的桌面高度和三香炉均分布局。

该素材不宣称任何宗教、玄学或超自然功效。
