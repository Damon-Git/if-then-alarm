# 小窗香炉素材

本目录用于未来放置 macOS 小窗中的 Q 版香炉图层。

## 对应插槽

- `censer/compact/lid`
- `censer/compact/mouth`
- `censer/compact/ash`
- `censer/compact/body`
- `censer/compact/feet`

## 未来素材

- `body.png`：Q 版香炉主体。
- `lid.png`：Q 版香炉盖。
- `mouth.png`：炉口。
- `ash.png`：香灰层。
- `feet.png`：底足。

小窗素材可以比主祭台更克制可爱，但必须保持与主祭台状态一一对应。
