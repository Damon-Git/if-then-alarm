# 小窗线香素材

本目录用于未来放置 macOS 小窗中的线香图层。

## 对应插槽

- `incense/compact/stick`
- `incense/compact/ash`
- `incense/compact/ember`
- `incense/compact/smoke`

## 未来素材

- `stick.png`：小窗香体。
- `ash.png`：小窗香灰。
- `ember.png`：小窗火星。
- `smoke.png`：小窗烟雾。

小窗线香需要优先保证在窄窗口中清楚可读，不追求复杂细节。

## 当前不做

- 不放真实线香图片。
- 不做燃烧、香灰掉落、火星闪动或烟雾动画。
