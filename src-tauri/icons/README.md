# 图标资产说明

当前目录用于存放 Tauri 桌面外壳需要的图标资产。现阶段只整理边界，不制作正式图标。

## 当前资产

- `icon.png`：临时占位应用图标，只用于让 Tauri dev/build 流程可以通过。

该图标不是正式品牌图标，不代表最终视觉方向，也不应作为后续 Dock、打包安装包或通知图标的设计基准。

## 当前菜单栏入口

当前 macOS 菜单栏右侧状态区使用文字“令”作为临时入口。

这样做是为了先验证窗口显示/隐藏、计时不中断、通知不中断等桌面行为。它不是正式菜单栏图标方案。

## 未来图标拆分

未来至少需要拆分这些图标资产：

- 应用图标：用于 Dock、应用包、安装包和系统应用列表。
- 菜单栏 template icon：用于 macOS 右侧状态区，应该适配深色/浅色模式。
- 通知图标：用于系统通知展示，必要时可沿用应用图标但应单独验收。

不要把应用图标、菜单栏图标和通知图标混用为同一个未验证素材。

## 未来菜单栏图标原则

- 使用单色 template icon，优先保证 macOS 状态栏可读性。
- 图形应简洁，避免复杂香炉、符箓纹样或小字。
- 可以从“令”字、香炉剪影或线香形态中提炼，但必须先做小尺寸可读性验收。
- 不在当前阶段生成或引入复杂图标。

## 当前不做

- 不制作正式应用图标。
- 不制作正式菜单栏 template icon。
- 不制作通知图标。
- 不生成完整 macOS iconset。
- 不接 DMG、签名、公证和正式发布流程。
