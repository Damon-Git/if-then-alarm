# 应用图标

本目录用于应用图标资产。

## 用途

- Dock 图标。
- `.app` 图标。
- 应用包和系统应用列表图标。
- Tauri 默认窗口图标。

## 当前文件

- `app-icon-v1.png`：当前应用图标 v1 源图，用于 Dock、`.app`、应用包和默认应用识别。
- `app-icon-v1@2x.png`：与源图内容一致的 Retina 打包副本。
- `app-icon-user-confirmed-source.png`：用户确认采用的新图标原图，用于后续重新导出。
- `placeholder-icon.png`：旧临时占位图标，保留用于历史追溯，不再接入 bundle。

`app-icon-v1.png` 是 `1024px × 1024px` RGBA PNG。当前视觉方向为用户确认的新图标：米纸底、山水、月形和亮金聚焦符箓线稿，不再使用大红背景和高光香炉。Tauri bundler 通过文件名中的 `@2x` 判断 Retina 密度，因此 `src-tauri/tauri.conf.json` 把同内容的 `app-icon-v1@2x.png` 配置到 `bundle.icon`。菜单栏 tray 使用独立的透明单色 template icon，不复用该应用图标。

## 视觉风格

应用图标的视觉关键词是清幽、朴素、仪式感和小尺寸识别。它表达“符箓 + 山水 + 月形 + 一炷香”的行动仪式，不表达神像、法器炫光或强烈宗教装饰。

- 主识别点：中心符箓线稿。符箓使用亮金聚焦色，在 Dock、Launchpad、应用程序列表等小尺寸下必须第一眼可见。
- 辅助意象：淡墨山水、右上月形、下方香与供台。辅助元素只提供氛围，不抢过符箓主线。
- 色彩基调：米纸暖白为底，山水和月形使用低饱和青灰绿，符箓使用高亮金色，香和印章使用克制赭金。
- 造型语言：大块留白、中心单一主符号、细线但高对比；不依赖细小文字、复杂纹样或写实材质。
- 禁用方向：大红底、金属高光香炉、火焰/闪光/镜头光晕、厚重拟物、Q 版角色、神像人物、复杂庙宇装饰。

后续微调必须同时查看 `64px`、`160px` 和 `1024px` 三档预览。若小尺寸下符箓不清楚，优先提升符箓亮度、对比和线条权重，而不是增加背景装饰。

## 当前接入

- `bundle.icon` 已切换到 `icons/app-icon/app-icon-v1@2x.png`。
- `default_window_icon()` 到 tray icon 的临时回退已移除，避免把彩色应用图标当作菜单栏 template icon。
- 内部 `.app` 阶段需要确认构建产物中的 `.icns`、Dock、`.app` 和系统通知识别表现。
- 完整 iconset 和其他平台输出留到正式发布准备阶段生成。

## 验收标准

- 至少覆盖 macOS 常用 iconset 尺寸，包含 16、32、64、128、256、512 和 1024 规格。
- Dock 中小尺寸可读，不依赖细小文字。
- `64px` 和 `160px` 预览中，亮金符箓应作为最强视觉识别点。
- 深色和浅色桌面背景下都能识别。
- 与菜单栏 template icon 分开验收。
- 与通知图标分开验收。
